Dự án lớn nhất — Hệ thống quản lý học tập (Learning Management System) với đầy đủ tính năng: khóa học, học viên, bài giảng, điểm số, chứng chỉ, thanh toán.

## Tổng quan tính năng

| Vai trò | Tính năng |
|---|---|
| **Admin** | Quản lý khóa học, users, báo cáo doanh thu, settings |
| **Giáo viên** | Tạo bài giảng, chấm bài, xem danh sách học viên |
| **Học viên** | Đăng ký khóa học, xem bài giảng, làm bài tập, xem điểm |

## Cloud Functions

| Function | Trigger | Mô tả |
|---|---|---|
| `onEnrollmentCreated` | Firestore onCreate | Gửi email xác nhận đăng ký |
| `onLessonCompleted` | Firestore onUpdate | Cập nhật progress enrollment |
| `onGradeCreated` | Firestore onCreate | Kiểm tra điều kiện cấp chứng chỉ |
| `dailyReminder` | Scheduled 8:00 AM | Nhắc học viên có bài mới |
| `weeklyReport` | Scheduled Monday 8:00 | Tổng hợp tuần cho admin |
| `sendEmail` | HTTP | Gửi email qua Resend |

## Quy trình xây dựng

<Steps>
  <Step title="Phase 1 — Setup & Core">
    Setup dự án Next.js + Firebase + shadcn/ui. Tạo cấu trúc thư mục, authentication, role-based routing.
  </Step>
  <Step title="Phase 2 — Course Management">
    CRUD Courses, Lessons. Upload video lên Cloud Storage. Hiển thị danh sách khóa học công khai.
  </Step>
  <Step title="Phase 3 — Enrollment & Learning">
    Đăng ký khóa học,.progress tracking, quiz/test.
  </Step>
  <Step title="Phase 4 — Grading & Certificates">
    Chấm bài, hệ thống điểm, tự động cấp chứng chỉ khi đạt điều kiện.
  </Step>
  <Step title="Phase 5 — Payment">
    Tích hợp VNPay/MoMo, webhook xác nhận thanh toán.
  </Step>
  <Step title="Phase 6 — Email & Notifications">
    Email templates, Cloud Functions trigger, daily reminders.
  </Step>
  <Step title="Phase 7 — AI Features">
    Tạo câu hỏi tự động, chấm bài essay, chatbot hỗ trợ.
  </Step>
  <Step title="Phase 8 — Security & Deploy">
    Security Rules chi tiết, deploy Vercel, test production.
  </Step>
</Steps>

## Security Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    match /courses/{courseId} {
      allow read: if resource.data.status == 'published'
               || request.auth != null;
      allow write: if get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'
                || get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'teacher';
    }

    match /enrollments/{enrollmentId} {
      allow read: if request.auth.uid == resource.data.studentId
               || get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
      allow create: if request.auth != null;
    }

    match /grades/{gradeId} {
      allow read: if request.auth.uid == resource.data.studentId
               || get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role in ['admin','teacher'];
      allow write: if get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role in ['admin','teacher'];
    }
  }
}
```

<Tip>
Dự án LMS rất lớn — mỗi phase nên hoàn thành trước khi sang phase tiếp. Kiểm tra kỹ từng phần trước khi tích hợp.
</Tip>
