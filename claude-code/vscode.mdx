Visual Studio Code (VS Code) là **trình soạn thảo code** — nơi bạn đọc và chỉnh sửa code. Với Claude Code, bạn chỉ cần mở file/dự án, đọc code do Claude Code tạo ra — VS Code sẽ tự động hỗ trợ format, màu sắc, gợi ý cơ bản.

## Cài đặt VS Code

<Steps>
  <Step title="Tải VS Code">
    1. Mở trình duyệt và truy cập https://code.visualstudio.com/
    2. Nhấn nút **Download for Windows** (hoặc macOS/Linux tùy theo máy bạn)
  </Step>
  <Step title="Cài đặt">
    1. Mở file vừa tải xuống
    2. Nhấn **Next** nhiều lần để cài đặt
    3. Giữ cấu hình mặc định
  </Step>
</Steps>

<Tip>
File tải về thường nằm trong thư mục Downloads, có tên như `VSCodeSetup.exe`.
</Tip>

## Trang trí VS Code (Themes & Icons)

### Themes — Thay đổi giao diện

Theme giúp VS Code đẹp hơn với màu sắc và phong cách khác nhau.

1. Nhấn `Ctrl + K Ctrl + T` (hoặc vào **File > Preferences > Color Theme**)
2. Chọn theme có sẵn hoặc nhấn **Browse Additional Themes**
3. Chọn theme bạn thích và nhấn **Enter**

| Theme | Phong cách | Ai nên dùng |
|---|---|---|
| **One Dark Pro** | Tối, màu xanh dịu | Người thích màu xanh |
| **Dracula Official** | Tối, nhiều màu | Phong cách retro/cyberpunk |

### File Icons — Biểu tượng cho file

Icon giúp phân biệt các loại file nhanh hơn trong cây thư mục.

1. Nhấn `Ctrl + Shift + X` để mở Extensions
2. Gõ `file icon` hoặc một trong các tên bên dưới
3. Nhấn **Install**

| Icon Set | Mô tả |
|---|---|
| **Material Icon Theme** | Biểu tượng vuông, nhiều màu |

### Cài Live Server (mở file HTML trực tiếp)

1. Mở Extensions (`Ctrl + Shift + X`)
2. Tìm `Live Server`
3. Nhấn **Install**
