## Tổng quan tính năng

| Tính năng | Mô tả |
|---|---|
| **Courses** | Khóa học theo level (A1→C2), theo chủ đề |
| **Lessons** | Video bài giảng, text content, audio listening |
| **Exercises** | Trắc nghiệm, điền blank, sắp xếp câu, nghe viết |
| **AI Chat** | Chatbot AI hỗ trợ giải thích ngữ pháp, từ vựng |
| **Flashcards** | Hệ thống flashcards spaced repetition |
| **Progress** | Theo dõi tiến độ, streak, thời gian học |
| **Achievements** | Huy hiệu, badges khi đạt mốc |

## Flashcards — Spaced Repetition (Thuật toán SM-2)

```typescript
// services/flashcards.ts
export function calculateNextReview(card: Flashcard, quality: number) {
  // quality: 0-5 (0 = hoàn toàn quên, 5 = nhớ rất tốt)

  let { easeFactor, interval } = card

  if (quality < 3) {
    // Nhớ sai → reset interval
    interval = 1
  } else {
    // Nhớ đúng → tăng interval
    if (interval === 1) interval = 6
    else interval = Math.round(interval * easeFactor)
  }

  // Cập nhật ease factor
  easeFactor = easeFactor + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02))
  if (easeFactor < 1.3) easeFactor = 1.3

  const nextReview = new Date()
  nextReview.setDate(nextReview.getDate() + interval)

  return { easeFactor, interval, nextReview }
}
```

## AI Chat — Cloud Function

```typescript
// functions/src/api/aiChat.ts
import { onRequest } from 'firebase-functions/v2/https'

export const aiChat = onRequest(async (req, res) => {
  const { userId, messages, courseId } = req.body

  // Kiểm tra giới hạn 20 tin/ngày
  const dailyCount = await getDailyMessageCount(userId)
  if (dailyCount >= 20) {
    return res.status(429).json({
      error: 'Bạn đã hết 20 tin nhắn hôm nay. Hãy quay lại ngày mai!'
    })
  }

  // Lấy context từ khóa học
  const courseSummary = await getCourseSummary(courseId)

  // Gọi AI API với system prompt giới hạn phạm vi
  const response = await callAI({
    system: `Bạn là trợ lý dạy tiếng Anh. Trả lời dựa trên nội dung khóa học: ${courseSummary}. Trả lời ngắn gọn, dễ hiểu bằng tiếng Việt.`,
    messages
  })

  // Lưu lịch sử chat
  await saveChatHistory(userId, messages, response)

  return res.json({ reply: response })
})
```

## Flashcard Component mẫu

```tsx
// components/Flashcard.tsx
'use client'
import { useState } from 'react'

export function Flashcard({ card, onRate }) {
  const [flipped, setFlipped] = useState(false)

  return (
    <div
      className="w-80 h-48 cursor-pointer perspective-1000"
      onClick={() => setFlipped(!flipped)}
    >
      <div className={`card-inner ${flipped ? 'flipped' : ''}`}>
        <div className="card-front flex items-center justify-center text-2xl font-bold">
          {card.word}
        </div>
        <div className="card-back flex flex-col items-center justify-center">
          <p className="text-lg">{card.meaning}</p>
          <p className="text-sm text-gray-500 italic mt-2">{card.example}</p>
        </div>
      </div>

      {flipped && (
        <div className="flex justify-center gap-2 mt-4">
          <Button onClick={() => onRate(0)} variant="destructive">Quên 😢</Button>
          <Button onClick={() => onRate(3)} variant="outline">Khó 😅</Button>
          <Button onClick={() => onRate(5)} variant="default">Dễ 😎</Button>
        </div>
      )}
    </div>
  )
}
```
