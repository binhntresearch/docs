Trước khi bắt đầu, bạn cần cài đặt các công cụ sau.

<Steps>
  <Step title="Visual Studio Code">
    Trình soạn thảo code miễn phí, phổ biến nhất hiện nay.

    Truy cập: [https://code.visualstudio.com](https://code.visualstudio.com)
  </Step>
  <Step title="Git & Bash Terminal">
    Hệ thống quản lý phiên bản phân tán và shell dòng lệnh mạnh mẽ.

    Truy cập: [https://git-scm.com/downloads](https://git-scm.com/downloads)
  </Step>
  <Step title="GitHub Desktop">
    Ứng dụng quản lý mã nguồn trên GitHub.

    Truy cập: [https://desktop.github.com](https://desktop.github.com)
  </Step>
  <Step title="Google Antigravity (Không bắt buộc)">
    Editor tương tự VS Code với tính năng hỗ trợ AI trong phát triển ứng dụng.

    Truy cập: [https://antigravity.google.com](https://antigravity.google.com)
  </Step>
</Steps>

<Note>
Google Antigravity là công cụ tùy chọn — bạn vẫn có thể hoàn thành khóa học mà không cần cài đặt.
</Note>
