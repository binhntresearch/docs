Bài thực hành giúp bạn tích hợp gửi email vào ứng dụng Next.js thông qua API Routes — sử dụng Resend, một dịch vụ email hiện đại.

## Mục tiêu

<Check>Tích hợp Resend SDK vào Next.js · Tạo email template HTML đẹp · Gửi email qua API Route · Trigger email tự động qua Cloud Functions khi có sự kiện</Check>

## Quy trình thực hành

<Steps>
  <Step title="Đăng ký Resend">
    Vào [resend.com](https://resend.com) → Đăng ký → Verify domain → Lấy API key.
  </Step>
  <Step title="Cài đặt SDK">
    ```bash
    npm install resend
    ```
  </Step>
  <Step title="Tạo email template">
    Sử dụng React Email để viết template:
    ```bash
    npm install @react-email/components
    ```
  </Step>
  <Step title="Tạo API Route gửi email">
    API tại `/api/send-email/enrollment-confirm`.
  </Step>
  <Step title="Tạo Cloud Function trigger">
    Khi user đăng ký khóa học → tự động gửi email xác nhận.
  </Step>
</Steps>

    Cloud Function: scheduled/paymentReminder

## Code mẫu — API Route

```typescript
// app/api/send-email/enrollment-confirm/route.ts
import { Resend } from 'resend'
import { EnrollmentConfirmEmail } from '@/emails/EnrollmentConfirm'

const resend = new Resend(process.env.RESEND_API_KEY)

export async function POST(req: Request) {
  const { studentEmail, studentName, courseName, startDate } = await req.json()

  try {
    const { data, error } = await resend.emails.send({
      from: 'LMS <noreply@lms.example.com>',
      to: [studentEmail],
      subject: `Chào mừng ${studentName} đến với khóa học ${courseName}!`,
      react: EnrollmentConfirmEmail({
        studentName,
        courseName,
        startDate,
      }),
    })

    if (error) return Response.json({ error }, { status: 400 })
    return Response.json({ success: true, data })
  } catch (error) {
    return Response.json({ error }, { status: 500 })
  }
}
```

## Email template mẫu — React Email

```tsx
// emails/EnrollmentConfirm.tsx
import { Html, Heading, Text, Button, Section } from '@react-email/components'

export function EnrollmentConfirmEmail({ studentName, courseName, startDate }) {
  return (
    <Html>
      <Section style={{ padding: '20px' }}>
        <Heading>Chào {studentName}!</Heading>
        <Text>
          Chào mừng bạn đến với khóa học <strong>{courseName}</strong>.
        </Text>
        <Text>Khóa học bắt đầu vào: <strong>{startDate}</strong></Text>
        <Button href="https://zoom.us/j/...">
          Tham gia buổi học
        </Button>
      </Section>
    </Html>
  )
}
```
