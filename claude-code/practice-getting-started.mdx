Bài thực hành đầu tiên giúp bạn làm quen với Claude Code, từ việc mở dự án, đặt câu hỏi đầu tiên, cho đến tạo một dự án hoàn chỉnh chạy được trên máy.

## Mục tiêu bài học

<Check>Sau bài này bạn sẽ biết cách mở Claude Code trong VS Code · Đặt prompt đầu tiên để Claude Code tạo file · Chạy dự án Next.js cơ bản trên máy</Check>

## Quy trình thực hành

<Steps>
  <Step title="Mở VS Code và Claude Code Panel">
    Nhấn `Ctrl+Shift+C` (Windows/Linux) hoặc `Cmd+Shift+C` (macOS) để mở Claude Code Panel.
  </Step>
  <Step title="Tạo thư mục dự án">
    Tạo folder rỗng tên `my-first-app` và mở trong VS Code.
  </Step>
  <Step title="Yêu cầu Claude Code khởi tạo dự án">
    Gõ prompt:

    ```
    Tạo dự án Next.js 15 trong thư mục này với TypeScript,
    Tailwind CSS 4, App Router. Không cần Firebase.
    Sau khi tạo xong, hướng dẫn tôi chạy dự án.
    ```
  </Step>
  <Step title="Chạy thử dự án">
    Claude Code sẽ chạy `npm install` và `npm run dev`. Mở `http://localhost:3000` để xem kết quả.
  </Step>
  <Step title="Sửa nội dung trang chủ">
    Yêu cầu Claude Code đổi nội dung trang chủ thành lời chào cá nhân của bạn.
  </Step>
</Steps>

## Bài tập mở rộng

<Tip>
Thử thay đổi màu sắc, font chữ, hoặc thêm một section mới cho landing page. Để Claude Code làm và quan sát cách nó tổ chức code.
</Tip>

## Kết quả mong đợi

Sau bài này bạn sẽ có:
- Một dự án Next.js 15 chạy được trên máy
- Trang chủ tùy chỉnh theo ý bạn
- Kinh nghiệm đầu tiên với Claude Code trong VS Code
