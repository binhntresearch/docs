Hướng dẫn chi tiết kiến trúc ứng dụng web hiện đại: **Frontend + Backend serverless** — sử dụng Next.js cho giao diện và Firebase cho toàn bộ hạ tầng phía sau.

## Kiến trúc phần mềm là gì?

Kiến trúc phần mềm là bản thiết kế tổng thể mô tả cách các thành phần của ứng dụng được tổ chức và giao tiếp với nhau. Giống như bản vẽ kiến trúc nhà — bạn cần biết phòng khách ở đâu, nhà bếp nối với phòng ăn thế nào — trước khi bắt tay xây.

## Tổng quan kiến trúc: 2 tầng chính

| Tầng | Vai trò | Công nghệ | Ví dụ cụ thể |
|---|---|---|---|
| **Frontend** | Giao diện người dùng | Next.js | Trang đăng nhập, dashboard, form tạo task |
| **Backend** | Xử lý logic, bảo mật | Firebase (Cloud Functions, Auth) | Xác thực user, gửi email thông báo |
| **Database** | Lưu trữ dữ liệu | Firebase (Firestore) | Danh sách tasks, users, projects |

## Sơ đồ kiến trúc tổng thể

```

│                    KIẾN TRÚC ỨNG DỤNG                       │

│  ├── Server Components (SEO, tốc độ)                       │
│  ├── Client Components (tương tác, realtime)               │

│  ☁️  BACKEND & DATABASE (Firebase + Google Cloud)           │
│  ├── Firestore (database NoSQL, realtime)                  │
│  ├── Authentication (đăng nhập/đăng ký)                    │
│  ├── Cloud Storage (lưu file)                              │
│  └── Cloud Functions (logic tự động)                       │
└─────────────────────────────────────────────────────────────┘
```

## Cách 2 tầng giao tiếp với nhau

Frontend (Next.js) và Backend (Firebase) hoạt động độc lập nhưng giao tiếp qua **Firebase SDK** — bộ công cụ cài trong dự án Next.js.

### Ví dụ: Luồng tạo task mới

1. **Người dùng mở form** — Bấm "Thêm task" → Dialog hiện ra với form nhập liệu
2. **Người dùng điền thông tin** — Nhập tiêu đề, mô tả, chọn người phụ trách, deadline
3. **Gửi dữ liệu lên Firebase** — Bấm "Lưu" → Firebase SDK gọi `addDoc()` ghi vào Firestore
4. **Firestore lưu và phát sự kiện** — Document mới được lưu, phát sự kiện realtime đến mọi thiết bị
5. **Giao diện cập nhật tự động** — Bảng danh sách cập nhật ngay lập tức, không reload
6. **Cloud Function chạy tự động** (nếu cấu hình) — Gửi email/notification cho người phụ trách

<Tip>
Frontend lo giao diện, Firebase lo backend. Bạn mô tả yêu cầu cho Claude Code — nó tự xây dựng cả 2 phần.
</Tip>
