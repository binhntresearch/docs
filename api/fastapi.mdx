## CRUD Endpoint Generation

```
Tạo Product CRUD endpoints với FastAPI:
- GET /api/v1/products (list + filter by category, search by name, pagination)
- GET /api/v1/products/{id}
- POST /api/v1/products (admin only)
- PUT /api/v1/products/{id}
- DELETE /api/v1/products/{id}

Stack:
- SQLAlchemy async
- Pydantic schemas với validation
- Depends(get_db) và Depends(get_current_admin)
- Proper status codes: 201 create, 204 delete, 404 not found
- Error handlers: 422 validation, 401 auth, 403 forbidden
- Include response models
- Add pagination schema (page, per_page, total, total_pages)
```

## Generated Code Example

```python
# app/routers/products.py — Output từ Claude Code
from typing import List, Optional
from fastapi import APIRouter, Depends, HTTPException, Query, status
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select, func, or_

from app.deps import get_db, get_current_user, require_admin
from app.models.product import Product
from app.models.user import User
from app.schemas.product import (
    ProductCreate, ProductUpdate, ProductResponse, ProductList
)
from app.schemas.common import PaginationParams, PaginatedResponse
from app.services.product_service import ProductService

router = APIRouter(prefix="/api/v1/products", tags=["Products"])


@router.get("/", response_model=PaginatedResponse[ProductResponse])
async def list_products(
    page: int = Query(1, ge=1),
    per_page: int = Query(20, ge=1, le=100),
    category: Optional[str] = None,
    search: Optional[str] = None,
    db: AsyncSession = Depends(get_db)
):
    """List products với pagination, filter by category, search by name."""
    service = ProductService(db)
    products, total = await service.list_products(
        page=page, per_page=per_page, category=category, search=search
    )
    return PaginatedResponse(
        data=products, page=page, per_page=per_page,
        total=total, total_pages=(total + per_page - 1) // per_page
    )


@router.get("/{product_id}", response_model=ProductResponse)
async def get_product(product_id: int, db: AsyncSession = Depends(get_db)):
    service = ProductService(db)
    product = await service.get_by_id(product_id)
    if not product:
        raise HTTPException(status_code=404, detail="Product not found")
    return product


@router.post("/", response_model=ProductResponse, status_code=status.HTTP_201_CREATED)
async def create_product(
    data: ProductCreate,
    db: AsyncSession = Depends(get_db),
    _: User = Depends(require_admin)
):
    service = ProductService(db)
    return await service.create(data)


@router.put("/{product_id}", response_model=ProductResponse)
async def update_product(
    product_id: int,
    data: ProductUpdate,
    db: AsyncSession = Depends(get_db),
    current_user: User = Depends(get_current_user)
):
    service = ProductService(db)
    product = await service.update(product_id, data, current_user)
    if not product:
        raise HTTPException(status_code=404, detail="Product not found")
    return product


@router.delete("/{product_id}", status_code=status.HTTP_204_NO_CONTENT)
async def delete_product(
    product_id: int,
    db: AsyncSession = Depends(get_db),
    _: User = Depends(require_admin)
):
    service = ProductService(db)
    success = await service.delete(product_id)
    if not success:
        raise HTTPException(status_code=404, detail="Product not found")
```

## Pydantic Schemas

```
Tạo Pydantic schemas cho Product:
- ProductBase: name (min 1, max 200), description (optional),
  category (enum), price (decimal), stock (>= 0)
- ProductCreate: kế thừa ProductBase
- ProductUpdate: tất cả fields optional, chỉ update khi có
- ProductResponse: + id, created_at, updated_at
- ProductList: cho response có products array
- Config from_attributes=True cho SQLAlchemy ORM
- Field validators cho business rules
```

## Authentication Setup

```
Tạo auth system với FastAPI:
- POST /api/v1/auth/register (email, password, name)
- POST /api/v1/auth/login (return access + refresh tokens)
- POST /api/v1/auth/refresh (rotate tokens)
- POST /api/v1/auth/logout (revoke refresh token)
- POST /api/v1/auth/forgot-password
- POST /api/v1/auth/reset-password
- JWT access token 15 min, refresh 7 days
- bcrypt password hashing
- Email service cho reset link
- Rate limit auth endpoints
```
