Phân tích dữ liệu (Data Analysis) là quá trình **kiểm tra, làm sạch, chuyển đổi và mô hình hóa dữ liệu** với mục tiêu khám phá thông tin hữu ích, đưa ra kết luận và hỗ trợ ra quyết định.

## Phân tích dữ liệu là gì?

<Tip>
Mọi quyết định kinh doanh ngày nay đều dựa trên dữ liệu. Từ việc chọn sản phẩm nào bán chạy, đến dự đoán doanh thu tháng sau, phân tích dữ liệu là nền tảng của mọi chiến lược.
</Tip>

## Hệ sinh thái thư viện

Các thư viện quan trọng bạn sẽ học trong khóa này:

| Thư viện | Mục đích | Icon |
|---|---|---|
| **NumPy** | Tính toán số học, mảng đa chiều | `calculator` |
| **Pandas** | Xử lý dữ liệu dạng bảng | `table` |
| **Matplotlib** | Vẽ biểu đồ cơ bản | `chart-bar` |
| **Seaborn** | Biểu đồ thống kê đẹp mắt | `chart-area` |
| **Jupyter** | Môi trường notebook tương tác | `notebook` |

## Quy trình phân tích dữ liệu

<Steps>
  <Step title="Thu thập dữ liệu">
    Lấy dữ liệu từ CSV, Excel, database, API, hoặc web scraping
  </Step>
  <Step title="Làm sạch dữ liệu">
    Xử lý missing values, loại bỏ duplicate, chuẩn hóa format
  </Step>
  <Step title="Khám phá dữ liệu (EDA)">
    Thống kê mô tả, phân phối, correlation giữa các biến
  </Step>
  <Step title="Trực quan hóa">
    Vẽ biểu đồ để tìm pattern và insight
  </Step>
  <Step title="Mô hình hóa & Dự đoán">
    Áp dụng ML/Statistical models (nâng cao)
  </Step>
  <Step title="Báo cáo & Triển khai">
    Trình bày kết quả cho stakeholders
  </Step>
</Steps>
