```python Vòng lặp
# Lặp qua danh sách
for item in [1, 2, 3]:
    print(item)

# Lặp với index
for i, item in enumerate(['a', 'b', 'c']):

while count < 3:

    count += 1
```

## Hàm

```python
def calculate_average(numbers):
    """Tính trung bình cộng của một list số."""
    if not numbers:
        return 0
    return sum(numbers) / len(numbers)

# Gọi hàm
avg = calculate_average([85, 92, 78])
print(f"Trung bình: {avg}")  # Trung bình: 85.0
```

## List Comprehension

```python
# Cách truyền thống
squares = []
for x in range(10):
    squares.append(x ** 2)

# List comprehension — gọn hơn
squares = [x ** 2 for x in range(10)]

# Có điều kiện
evens = [x for x in range(20) if x % 2 == 0]
```
