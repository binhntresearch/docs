## Common Workflow

```
Data Source
    │
    ├── Connect (auth, credentials)
    │
    ├── Extract (query, request, stream)
    │
    ├── Validate (schema, types, ranges)
    │
    ├── Transform (clean, enrich, aggregate)
    │
    └── Load (CSV, DB, DataFrame, Parquet)
```

## Best Practices

<Check>
- **Log metadata:** Source, fetch time, schema version
- **Version data:** DVC, LakeFS cho reproducibility
- **Handle rate limits:** Retry, backoff, caching
- **Secure credentials:** Environment variables, never in code
- **Validate early:** Fail fast on bad data
- **Document schemas:** JSON Schema, OpenAPI specs
</Check>
