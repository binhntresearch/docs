## Risk Management

```python
class RiskManager:
    def __init__(self, max_portfolio_risk=0.02, max_position_size=0.10):
        self.max_portfolio_risk = max_portfolio_risk  # 2% per trade
        self.max_position_size = max_position_size    # 10% max

    def calculate_position_size(self, account_balance, entry_price, stop_loss):
        risk_amount = account_balance * self.max_portfolio_risk
        price_risk = abs(entry_price - stop_loss)
        position_size = risk_amount / price_risk
        position_value = position_size * entry_price

        # Cap at max position size
        max_value = account_balance * self.max_position_size
        if position_value > max_value:
            position_size = max_value / entry_price

        return round(position_size, 4)
```

## Execution Agent

```python
# Trading bot với Coinbase API (demo, không phải real trading)
from coinbase import trading

class ExecutionAgent:
    def __init__(self, api_key, api_secret):
        self.client = trading.Tradeable(api_key, api_secret)

    def place_order(self, symbol, side, size, price=None):
        if price:
            # Limit order
            order = self.client.place_limit_order(
                product_id=f"{symbol}-USD",
                side=side,
                price=str(price),
                size=str(size)
            )
        else:
            # Market order
            order = self.client.place_market_order(
                product_id=f"{symbol}-USD",
                side=side,
                size=str(size)
            )
        return order

    def set_stop_loss(self, symbol, size, stop_price):
        return self.client.place_stop_order(
            product_id=f"{symbol}-USD",
            side="sell",
            size=str(size),
            stop_price=str(stop_price)
        )
```

## Best Practices

<Check>
- **Paper trading first:** Test trên testnet tối thiểu 30 ngày
- **Risk management:** Max 2% risk per trade, stop-loss bắt buộc
- **Backtesting:** Test trên historical data trước khi live
- **Monitoring:** Real-time alerts cho mọi trade
- **Kill switch:** Ability to instantly stop all trading
- **Logging:** Log mọi quyết định để audit
