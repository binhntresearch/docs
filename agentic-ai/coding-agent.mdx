Coding Agent tự động **viết code, chạy test, fix bugs** — như một junior developer hoạt động 24/7.

## Kiến trúc

```
User: "Build a REST API for user management"
    ↓
Planner Agent → Phân tích requirements
    ↓
┌── Code Agent → Viết code
├── Test Agent → Viết test
├── Review Agent → Code review
└── Fix Agent → Fix bugs nếu có
    ↓
Deploy Agent → Deploy lên server
```

## Test-Driven Coding Agent

```python
def tdd_agent(requirement):
    """Test-Driven Development agent."""
    messages = [{"role": "user", "content": f"Build using TDD: {requirement}"}]

    for _ in range(30):
        response = client.messages.create(
            model="claude-sonnet-5-20250514",
            system="Always write tests FIRST, then implementation. Run tests after each step.",
            tools=coding_tools,
            messages=messages,
            max_tokens=4096
        )

        if response.stop_reason == "end_turn":
            return response.content[0].text

        messages.append({"role": "assistant", "content": response.content})
        # Execute tools and continue
```

## Debugging Agent

```python
def debug_agent(error_message, code_context):
    messages = [{
        "role": "user",
        "content": f"""
Error: {error_message}

Code context:
{code_context}

Find and fix the bug. Explain what went wrong.
"""
    }]

    response = client.messages.create(
        model="claude-sonnet-5-20250514",
        tools=coding_tools,
        messages=messages,
        max_tokens=4096
    )
    return response.content[0].text
```

## Feature decomposition

```python
def implement_feature(feature_description):
    # 1. Break feature into tasks
    tasks = plan_tasks(feature_description)

    results = []
    for task in tasks:
        # 2. Implement each task
        code = write_code(task)

        # 3. Test
        test_result = run_tests(task)

        # 4. Fix if failed
        if test_result.failed:
            code = fix_code(code, test_result.errors)

        results.append({"task": task, "code": code, "status": "passed"})

    # 5. Integration test
    integration_result = run_integration_tests()
    return results
```

## Agent + CI/CD

```yaml
# GitHub Actions — auto-fix PR
name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review
        run: |
          python review_agent.py --diff "${{ github.event.pull_request.diff_url }}"
      - name: Auto-fix
        if: steps.review.outputs.has_critical
        run: |
          python fix_agent.py --diff "${{ github.event.pull_request.diff_url }}"
```
