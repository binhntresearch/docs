## Lộ Trình Học Tập

Khoá học được chia thành các phần từ cơ bản đến nâng cao. Bạn có thể học tuần tự hoặc chọn chủ đề phù hợp với nhu cầu.

<Steps>
  <Step title="Giới Thiệu & Cài Đặt">
    Tìm hiểu Claude Code là gì, cài đặt CLI và xác thực tài khoản. Chỉ mất 5–10 phút.
  </Step>
  <Step title="Sử Dụng Cơ Bản">
    Học cách bắt đầu session, giao tiếp với Claude, sử dụng các lệnh tắt và quản lý ngữ cảnh dự án với CLAUDE.md.
  </Step>
  <Step title="Tính Năng Nâng Cao">
    Khám phá subagents chạy song song, tích hợp MCP, hooks tự động hoá và hệ thống skills/plugins.
  </Step>
  <Step title="Tích Hợp Công Cụ">
    Kết nối Claude Code với VS Code, JetBrains, GitHub Actions và desktop app để tối ưu workflow.
  </Step>
</Steps>
