AI Safety & Ethics = đảm bảo AI systems **công bằng, minh bạch, có trách nhiệm** — không gây harm cho society.

## Common Bias Types

| Bias | Mô tả | Ví dụ |
|---|---|---|
| **Selection bias** | Training data không representative | Dataset chỉ có ảnh người da trắng → Face recognition sai với người da đen |
| **Confirmation bias** | Chọn data confirm hypothesis | Chỉ lấy data thành công → bỏ qua failures |
| **Historical bias** | Data reflects past discrimination | Hiring model learn gender bias từ historical hiring |
| **Measurement bias** | Different groups measured differently | Disease prediction tốt hơn ở group A vì data nhiều hơn |

## Fairness Metrics

```python
from fairlearn.metrics import (
    MetricFrame,
    demographic_parity_difference,
    equalized_odds_difference
)

# Evaluate fairness across groups
metric_frame = MetricFrame(
    metrics={

```

## Mitigating Bias

```python
from fairlearn.reductions import ExponentiatedGradient, DemographicParity

# Pre-processing: Resample data
from imblearn.over_sampling import SMOTE
smote = SMOTE(sampling_strategy='auto')
X_resampled, y_resampled = smote.fit_resample(X, y)

# In-processing: Add fairness constraint
mitigator = ExponentiatedGradient(
    estimator=base_estimator,
    constraints=DemographicParity()
)
mitigator.fit(X_train, y_train, sensitive_features=gender_train)
y_pred_fair = mitigator.predict(X_test)

# Post-processing: Adjust thresholds per group
from fairlearn.postprocessing import ThresholdOptimizer
threshold_optimizer = ThresholdOptimizer(
    estimator=base_estimator,
    constraints="demographic_parity"
)
threshold_optimizer.fit(X_train, y_train, sensitive_features=gender_train)
y_pred_fair = threshold_optimizer.predict(X_test, sensitive_features=gender_test)
```

## Explainability — SHAP & LIME

```python
# SHAP values
import shap

# Tree models
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X_test)

# Summary plot
shap.summary_plot(shap_values, X_test, feature_names=feature_names)

# Individual prediction explanation
shap.force_plot(
    explainer.expected_value,
    shap_values[0],
    X_test.iloc[0],
    feature_names=feature_names
)

# LIME — Local Interpretable Model-agnostic Explanations
from lime import lime_tabular

explainer = lime_tabular.LimeTabularExplainer(
    training_data=X_train.values,
    feature_names=feature_names,
    mode='classification'
)

explanation = explainer.explain_instance(
    X_test.iloc[0].values,
    model.predict_proba
)
explanation.show_in_notebook()
```

## Red Teaming

```python
# Test model for edge cases and adversarial inputs
def red_team_test(model, test_cases):
    failures = []
    for input_data, expected in test_cases:
        prediction = model.predict([input_data])
        if prediction != expected:
            failures.append({
                'input': input_data,
                'expected': expected,
                'predicted': prediction
            })
    return failures

# Test for fairness across demographics
fairness_failures = red_team_test(model, [
    # Same qualifications, different names
    ({'name': 'John Smith', 'years_exp': 5}, 'hire'),
    ({'name': 'Jamal Johnson', 'years_exp': 5}, 'hire'),
    # Should predict same outcome
])
```
