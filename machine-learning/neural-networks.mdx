Neural Networks là **foundation của Deep Learning** — inspired by brain neurons, learn complex patterns từ data.

## Perceptron — Single Neuron

```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def perceptron(X, weights, bias):
    """Single neuron: weighted sum + activation"""
    z = np.dot(X, weights) + bias
    return sigmoid(z)

# Example
X = np.array([1.0, 2.0, 3.0])
weights = np.array([0.5, -0.2, 0.8])
bias = 0.1

output = perceptron(X, weights, bias)
print(f"Output: {output:.4f}")  # Between 0 and 1
```

## Multi-Layer Perceptron (MLP)

```python
from sklearn.neural_network import MLPClassifier
from sklearn.datasets import load_digits
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# Load digits dataset (8x8 images, classify 0-9)
digits = load_digits()
X_train, X_test, y_train, y_test = train_test_split(
    digits.data, digits.target, random_state=42, stratify=digits.target
)

# 3-layer MLP
mlp = MLPClassifier(
    hidden_layer_sizes=(128, 64),  # 2 hidden layers
    activation='relu',
    solver='adam',
    learning_rate_init=0.001,
    max_iter=200,
    random_state=42
)

mlp.fit(X_train, y_train)
y_pred = mlp.predict(X_test)
print(f"MLP Accuracy: {accuracy_score(y_test, y_pred):.4f}")
```

## PyTorch — More Powerful

```python
import torch
import torch.nn as nn

# Define model
class SimpleNN(nn.Module):
    def __init__(self, input_size, hidden_size, num_classes):
        super().__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(hidden_size, hidden_size)
        self.fc3 = nn.Linear(hidden_size, num_classes)
        self.dropout = nn.Dropout(0.2)

    def forward(self, x):
        out = self.fc1(x)
        out = self.relu(out)
        out = self.dropout(out)
        out = self.fc2(out)
        out = self.relu(out)
        out = self.fc3(out)
        return out

# Instantiate
model = SimpleNN(input_size=784, hidden_size=128, num_classes=10)
print(model)
```

## Activation Functions

```python
import torch
import torch.nn.functional as F
import matplotlib.pyplot as plt
import numpy as np

z = torch.linspace(-5, 5, 100)

activations = {
    'ReLU': F.relu(z),
    'Sigmoid': torch.sigmoid(z),
    'Tanh': torch.tanh(z),
    'LeakyReLU': F.leaky_relu(z, negative_slope=0.1),
    'GELU': F.gelu(z)
}

for name, act in activations.items():
    plt.plot(z.numpy(), act.numpy(), label=name)

plt.legend()
plt.title("Activation Functions")
plt.grid(True)
plt.show()
```

| Activation | Range | Use case |
|---|---|---|
| **ReLU** | [0, ∞] | Hidden layers default |
| **Sigmoid** | [0, 1] | Binary classification output |
| **Tanh** | [-1, 1] | Centered data, RNNs |
| **LeakyReLU** | (-∞, ∞) | Avoid dying ReLU |
| **GELU** | (-∞, ∞) | Transformers (BERT, GPT) |
| **Softmax** | [0, 1] | Multi-class output |

## Backpropagation — How NN Learn

```
Forward pass:
  Input → Layer 1 → Layer 2 → Layer 3 → Output → Loss

Backward pass:
  Compute gradients ∂Loss/∂weights
  Update weights: w ← w - lr × ∂Loss/∂w
```

```python
import torch

# Simple example
x = torch.tensor([1.0, 2.0])
y_true = torch.tensor([1.0])

# Forward
w = torch.tensor([0.5, 0.5], requires_grad=True)
y_pred = (x * w).sum()

# Loss
loss = (y_pred - y_true) ** 2
print(f"Loss: {loss.item():.4f}")

# Backward (automatic differentiation)
loss.backward()
print(f"Gradients: {w.grad}")

# Update
with torch.no_grad():
    w -= 0.1 * w.grad

print(f"Updated weights: {w}")
```

## Training Loop

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

# Data
X = torch.randn(1000, 784)
y = torch.randint(0, 10, (1000,))

train_dataset = TensorDataset(X, y)
train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True)

# Model + loss + optimizer
model = SimpleNN(784, 128, 10)
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# Training
for epoch in range(5):
    for batch_X, batch_y in train_loader:
        # Forward
        outputs = model(batch_X)
        loss = criterion(outputs, batch_y)

        # Backward + optimize
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

<Tip>
Modern frameworks (PyTorch, TensorFlow) tự động hóa backpropagation qua `loss.backward()` + `optimizer.step()`. Bạn chỉ cần define architecture, loss function, và optimizer — phần còn lại framework lo.
</Tip>
