Edge AI = chạy ML models **trên thiết bị** (phone, Raspberry Pi, IoT) thay vì cloud — offline, nhanh hơn, privacy hơn.

## Export Models cho Edge

### ONNX — Universal Format

```python
import torch

# PyTorch → ONNX
dummy_input = torch.randn(1, 3, 224, 224)
torch.onnx.export(model, dummy_input, "model.onnx", opset_version=11)

# Optimized ONNX
import onnxruntime as ort

session = ort.InferenceSession(
    "model.onnx",
    providers=['CUDAExecutionProvider', 'CPUExecutionProvider']
)
```

### TensorFlow Lite — Mobile

```python
import tensorflow as tf

# Convert Keras model to TFLite
converter = tf.lite.TFLiteConverter.from_keras_model(model)
converter.optimizations = [tf.lite.Optimize.DEFAULT]  # Quantize
tflite_model = converter.convert()

with open('model.tflite', 'wb') as f:
    f.write(tflite_model)
```

### CoreML — iOS/macOS

```python
import coremltools as ct

# Convert PyTorch to CoreML
mlmodel = ct.convert(model, inputs=[ct.ImageType(shape=(1, 3, 224, 224))])
mlmodel.save("model.mlmodel")
```

## Quantization — Reduce Model Size

```python
import torch.quantization

# Post-training quantization
model_quantized = torch.quantization.quantize_dynamic(
    model,
    {torch.nn.Linear},  # Quantize linear layers
    dtype=torch.qint8
)

# Reduce model size ~4x, minor accuracy loss
```

## Edge Deployment Examples

### Mobile — React Native + ONNX

```python
# onnxruntime-react-native
import ONNXRuntime from 'onnxruntime-react-native';

const session = await ONNXRuntime.createSession('model.onnx');
const results = await session.run({ input: inputData });
```

### Raspberry Pi — TFLite

```python
import tflite_runtime.interpreter as tflite
import numpy as np

# Load model
interpreter = tflite.Interpreter(model_path='model.tflite')
interpreter.allocate_tensors()

input_details = interpreter.get_input_details()
output_details = interpreter.get_output_details()

# Run inference
input_data = np.array(image, dtype=np.float32)
interpreter.set_tensor(input_details[0]['index'], input_data)
interpreter.invoke()
output = interpreter.get_tensor(output_details[0]['index'])
```

### Browser — TensorFlow.js

```javascript
import * as tf from '@tensorflow/tfjs';

// Load model
const model = await tf.loadLayersModel('model.json');

// Predict
const tensor = tf.browser.fromPixels(image)
  .resizeBilinear([224, 224])
  .expandDims()
  .div(255.0);

const prediction = model.predict(tensor);
console.log(prediction.dataSync());
```

## Model Optimization

| Technique | Size Reduction | Speed Improvement |
|---|---|---|
| **Quantization (int8)** | 4x | 2-4x faster |
| **Pruning** | 2-10x | 1.5-3x faster |
| **Knowledge Distillation** | Varies | Varies |
| **ONNX Optimization** | 1.5-2x | 1.5-2x faster |

## Edge vs Cloud

<Tabs>
  <Tab title="Edge (On-device)">
    - ✅ Low latency (< 100ms)
    - ✅ Works offline
    - ✅ Better privacy
    - ❌ Limited compute power
    - ❌ Limited storage
    - ❌ Model updates harder
  </Tab>
  <Tab title="Cloud">
    - ✅ Unlimited compute
    - ✅ Easy model updates
    - ✅ Large models possible
    - ❌ Network latency (200ms+)
    - ❌ Requires internet
    - ❌ Data privacy concerns
  </Tab>
</Tabs>

<Tip>
Edge AI phù hợp: computer vision on phones, IoT sensors, voice assistants, real-time apps cần latency thấp. Cloud phù hợp: complex reasoning, large models, batch processing.
</Tip>
