## Linear Algebra

### Vectors & Matrices

```python
import numpy as np

# Vector
v = np.array([1, 2, 3])

# Matrix
A = np.array([[1, 2], [3, 4]])

# Shape
print(A.shape)  # (2, 2)

# Matrix multiplication
B = np.array([[5, 6], [7, 8]])
print(A @ B)  # [[19 22], [43 50]]

# Transpose
print(A.T)

# Dot product
print(np.dot(v, v))  # 14 (1² + 2² + 3²)
```

### Norms — khoảng cách

```python
# L1 norm — Manhattan distance
print(np.linalg.norm(v, ord=1))  # 6

# L2 norm — Euclidean distance
print(np.linalg.norm(v))  # 3.74

# Distance giữa 2 vectors
a = np.array([1, 2])
b = np.array([4, 6])
print(np.linalg.norm(a - b))  # 5.0
```

## Calculus

### Derivatives

```python
# Gradient của loss function
# f(x) = x² → f'(x) = 2x
import sympy as sp

x = sp.Symbol('x')
f = x ** 2
derivative = sp.diff(f, x)
print(derivative)  # 2*x

# Partial derivatives
y = sp.Symbol('y')
g = x ** 2 + y ** 2
print(sp.diff(g, x))  # 2*x
print(sp.diff(g, y))  # 2*y
```

### Gradient Descent

```python
# Gradient descent concept
def gradient_descent(learning_rate=0.01, iterations=100):
    x = 10.0  # Starting point

    for _ in range(iterations):
        gradient = 2 * x  # derivative of f(x) = x²
        x = x - learning_rate * gradient

    return x

print(gradient_descent())  # Close to 0 (minimum of x² is at x=0)
```

## Statistics

### Mean, Median, Variance

```python
import numpy as np

data = np.array([1, 2, 3, 4, 5, 100])  # Có outlier

# Mean (bị kéo bởi outlier)
print(np.mean(data))  # 19.17

# Median (robust)
print(np.median(data))  # 3.5

# Variance
print(np.var(data))  # 1340.47

# Standard deviation
print(np.std(data))  # 36.61
```

### Probability Distributions

```python
from scipy import stats

# Normal distribution
mean, std = 0, 1
x = stats.norm.pdf(0, mean, std)  # PDF at 0
print(f"PDF at 0: {x:.3f}")  # 0.399

# Sampling
samples = np.random.normal(100, 15, 1000)  # 1000 samples, mean=100, std=15
print(f"Sample mean: {np.mean(samples):.2f}")
print(f"Sample std: {np.std(samples):.2f}")
```

### Bayes' Theorem

```
P(A|B) = P(B|A) × P(A) / P(B)

Example: Spam detection
P(spam | contains "free") = P(contains "free" | spam) × P(spam) / P(contains "free")
```

## Linear Regression Math

```
Hypothesis:    ŷ = wx + b
Loss (MSE):   L = (1/n) Σ(ŷᵢ - yᵢ)²
Gradient:     ∂L/∂w = (2/n) Σ(ŷᵢ - yᵢ) × xᵢ
              ∂L/∂b = (2/n) Σ(ŷᵢ - yᵢ)
Update:       w ← w - α × ∂L/∂w
              b ← b - α × ∂L/∂b
```

## Logistic Regression Math

```
Hypothesis:    p = σ(z) = 1 / (1 + e^(-z))
              where z = wx + b
Loss (BCE):   L = -(1/n) Σ[y log(p) + (1-y) log(1-p)]
```

```python
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# Example
z = np.array([0.0, 1.0, -1.0, 5.0])
print(sigmoid(z))
# [0.5, 0.731, 0.269, 0.993]
```
