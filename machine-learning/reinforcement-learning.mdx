Reinforcement Learning (RL) = Agent học qua **trial-and-error** — thực hiện actions trong environment, nhận rewards/punishments, tối ưu strategy.

## RL Basics

```
Agent ←→ Environment
  │           │
  Action      State
  ↓           ↑
  Reward ←── Observation
```

| Concept | Mô tả |
|---|---|
| **Agent** | Learner — robot, game player, trading bot |
| **Environment** | World the agent operates in |
| **State** | Current situation |
| **Action** | What agent can do |
| **Reward** | Feedback signal (+/-) |
| **Policy** | Strategy: state → action mapping |

## Q-Learning (Tabular)

```python
import numpy as np
import gym

# Create environment
env = gym.make('FrozenLake-v1', is_slippery=False)

# Q-table
q_table = np.zeros((env.observation_space.n, env.action_space.n))

# Hyperparameters
alpha = 0.1    # Learning rate
gamma = 0.99   # Discount factor
epsilon = 0.1  # Exploration rate

# Training
for episode in range(10000):
    state = env.reset()
    done = False

    while not done:
        # Epsilon-greedy action selection
        if np.random.random() < epsilon:
            action = env.action_space.sample()
        else:
            action = np.argmax(q_table[state])

        next_state, reward, done, _ = env.step(action)

        # Q-update
        q_table[state, action] += alpha * (
            reward + gamma * np.max(q_table[next_state]) - q_table[state, action]
        )

        state = next_state
```

```python
import torch.nn as nn
import random

class DQN(nn.Module):
    def __init__(self, state_dim, action_dim):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(state_dim, 128),
            nn.ReLU(),
            nn.Linear(128, 128),
            nn.ReLU(),
            nn.Linear(128, action_dim)
        )

    def forward(self, x):
        return self.network(x)

# Experience replay
replay_buffer = deque(maxlen=10000)

def train_dqn(env, episodes=1000):
```

## Policy Gradient — PPO

```python
import torch
import torch.nn as nn

class PolicyNetwork(nn.Module):
    def __init__(self, state_dim, action_dim):
        super().__init__()
        self.actor = nn.Sequential(
            nn.Linear(state_dim, 128),
            nn.ReLU(),
            nn.Linear(128, action_dim),
            nn.Softmax(dim=-1)
        )

    def forward(self, state):
        return self.actor(state)

# PPO algorithm (simplified)
def ppo_step(policy, optimizer, states, actions, old_log_probs, rewards):
    # Compute advantages
    advantages = rewards - rewards.mean()

    # New log probs
    new_log_probs = torch.log(policy(states).gather(1, actions))

    # PPO clipped objective
    ratio = torch.exp(new_log_probs - old_log_probs)
    clipped = torch.clamp(ratio, 0.8, 1.2)
    loss = -torch.min(ratio * advantages, clipped * advantages).mean()

    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

```
1. Supervised Fine-tuning (SFT): Train model trên human demonstrations
```

## OpenAI Gym / Gymnasium

```python
import gymnasium as gym

# Create environment
env = gym.make('CartPole-v1')

# Run random agent
for episode in range(10):
    state, _ = env.reset()
    total_reward = 0
    done = False

    while not done:
        action = env.action_space.sample()
        state, reward, terminated, truncated, info = env.step(action)
        total_reward += reward
        done = terminated or truncated

    print(f"Episode {episode+1}: Total reward = {total_reward}")

env.close()
```

## When to Use RL?

<Check>
- ✅ **Games:** AlphaGo, Atari, Chess
- ✅ **Robotics:** Robot control, navigation
- ✅ **Trading:** Stock/crypto trading bots
- ✅ **RLHF:** Fine-tuning LLMs (ChatGPT, Claude)
- ❌ **Not needed:** Static prediction tasks (use supervised)
- ❌ **Not needed:** Simple classification/regression
</Check>

</Tip>
