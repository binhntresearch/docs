ML systems có **attack surface riêng** — từ training data đến deployed model đều có thể bị tấn công.

## Các mối đe dọa chính

| Mối đe dọa | Vector | Tác động |
|---|---|---|
| **Data poisoning** | Train data bị thay đổi | Model output sai |
| **Model stealing** | API bị query nhiều lần | IP theft |
| **Adversarial attacks** | Input crafted đặc biệt | Wrong predictions |
| **Membership inference** | Xác định data có trong training set | Privacy leak |
| **Backdoor attacks** | Trigger pattern ẩn trong model | Hidden behavior |
| **PII leakage** | Model memorize training data | GDPR violation |

### Detect Anomalies trong Training Data

```python
from sklearn.ensemble import IsolationForest

def detect_poisoned_samples(X_train, y_train, contamination=0.05):
    """Detect potential poisoned samples using anomaly detection"""
    iso = IsolationForest(contamination=contamination, random_state=42)
    preds = iso.fit_predict(X_train)
    # -1 = anomaly, 1 = normal
    poisoned_mask = preds == -1
    return poisoned_mask

# Remove suspicious samples
poisened_mask = detect_poisoned_samples(X_train, y_train)
X_clean = X_train[~poisened_mask]
y_clean = y_train[~poisened_mask]

print(f"Removed {poisened_mask.sum()} suspicious samples")
```

## Model Stealing Protection

### Rate Limiting cho API

```python
from fastapi import FastAPI, Request, HTTPException
from collections import defaultdict
import time
import hashlib

app = FastAPI()
user_queries = defaultdict(list)

@app.middleware("http")
async def rate_limit_predict(request: Request, call_next):
    user_id = request.headers.get('X-User-ID', request.client.host)
    now = time.time()

    user_queries[user_id] = [
        t for t in user_queries[user_id] if now - t < 3600
    ]

    # Max 100 predictions per hour per user
    if len(user_queries[user_id]) >= 100:
        raise HTTPException(429, "Rate limit: 100 predictions/hour")

    user_queries[user_id].append(now)
    return await call_next(request)
```

### Add Noise to Predictions

```python
import numpy as np

def noisy_predict(model, X, noise_scale=0.01):
    """Add small noise to prevent exact model extraction"""
    predictions = model.predict_proba(X)
    noise = np.random.normal(0, noise_scale, predictions.shape)
    noisy = predictions + noise
    return noisy / noisy.sum(axis=1, keepdims=True)  # Re-normalize
```

## Membership Inference Attack Defense

```python
# Differential Privacy training với PyTorch Opacus
from opacus import PrivacyEngine

def train_with_dp(model, train_loader, epochs=10, epsilon=1.0):
    """Train với differential privacy guarantee"""
    optimizer = torch.optim.Adam(model.parameters())

    privacy_engine = PrivacyEngine()
    model, optimizer, train_loader = privacy_engine.make_private(
        module=model,
        optimizer=optimizer,
        data_loader=train_loader,
        noise_multiplier=1.1,
        max_grad_norm=1.0,
    )

    for epoch in range(epochs):
        for X, y in train_loader:
            optimizer.zero_grad()
            loss = F.cross_entropy(model(X), y)
            loss.backward()
            optimizer.step()

    epsilon = privacy_engine.get_epsilon(delta=1e-5)
    print(f"Privacy budget (ε): {epsilon:.2f}")
    # ε < 1 = strong privacy, ε > 10 = weak privacy
```

## PII Protection trong Training Data

```python
import re

PII_PATTERNS = {
    'email': r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b',
    'phone': r'\b\d{10,11}\b',
    'ssn': r'\b\d{3}-\d{2}-\d{4}\b',
    'credit_card': r'\b\d{4}[-\s]?\d{4}[-\s]?\d{4}[-\s]?\d{4}\b',
}

def redact_pii(text: str) -> str:
    for label, pattern in PII_PATTERNS.items():
        text = re.sub(pattern, f'[REDACTED_{label.upper()}]', text)
    return text

# Apply to dataset
df['description_redacted'] = df['description'].apply(redact_pii)
print(df['description_redacted'].head())
```

## Model Serving Security

```python
class SecureModelServer:
    def __init__(self, model, api_key_manager):
        self.model = model
        self.api_keys = api_key_manager

    def predict(self, X, user_id):
        # 1. Authentication
        if not self.api_keys.validate(user_id):
            raise PermissionError("Invalid API key")

        # 2. Input validation
        X = self.validate_input(X)

        # 3. Prediction
        raw_output = self.model.predict(X)

        # 4. Output filtering
        safe_output = self.add_uncertainty_quantification(raw_output)

        # 5. Audit log
        self.log_prediction(user_id, X, safe_output)

        return safe_output

    def add_uncertainty_quantification(self, predictions):
        """Return predictions WITH confidence scores"""
        if hasattr(self.model, 'predict_proba'):
            proba = self.model.predict_proba(predictions)
            confidence = proba.max(axis=1)
            return predictions, confidence
        return predictions, np.ones(len(predictions))
```
