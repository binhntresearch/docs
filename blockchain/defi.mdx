## AMM — Automated Market Maker

<Callout type="info">
Uniswap-style AMM dùng constant product formula: `x * y = k`. Không cần order book — giá tự điều chỉnh dựa trên tỷ lệ pool.
</Callout>

```solidity
// Simplified AMM Contract
contract SimpleAMM {
    IERC20 public tokenA;
    IERC20 public tokenB;

    function swap(address tokenIn, uint amountIn) external returns (uint amountOut) {
        require(tokenIn == address(tokenA) || tokenIn == address(tokenB));

        uint reserveIn = IERC20(tokenIn).balanceOf(address(this));
        uint reserveOut = (tokenIn == address(tokenA))
            ? IERC20(tokenB).balanceOf(address(this))
            : IERC20(tokenA).balanceOf(address(this));

        // x * y = k
        amountOut = (amountIn * reserveOut) / (reserveIn + amountIn);

        IERC20(tokenIn).transferFrom(msg.sender, address(this), amountIn);
        IERC20(tokenIn == address(tokenA) ? tokenB : tokenA).transfer(msg.sender, amountOut);
    }

    function removeLiquidity(uint liquidity) external returns (uint amountA, uint amountB) {

}
```

## Lending Protocol

```solidity
contract LendingPool {
    mapping(address => uint) public deposits;
    mapping(address => uint) public borrows;
    uint public totalDeposits;
    uint public constant LTV = 75; // 75% Loan-to-Value

    // Deposit
    function deposit(uint amount) external {
        IERC20(token).transferFrom(msg.sender, address(this), amount);
        deposits[msg.sender] += amount;
        totalDeposits += amount;
    }

    // Borrow (need collateral)
    function borrow(uint amount) external {
        uint collateral = deposits[msg.sender] * LTV / 100;
        require(collateral >= borrows[msg.sender] + amount, "Undercollateralized");

        borrows[msg.sender] += amount;
        IERC20(token).transfer(msg.sender, amount);
    }

    // Repay
    function repay(uint amount) external {
        IERC20(token).transferFrom(msg.sender, address(this), amount);
        borrows[msg.sender] -= amount;
    }
}
```

## DeFi Risks

<Warning>
- **Smart contract bugs:** Contract bị hack = mất toàn bộ TVL
- **Oracle manipulation:** Giá feed bị thao túng
- **Rug pull:** Team bỏ chạy với tiền liquidity
- **Impermanent loss:** LP bị loss khi giá token biến động mạnh
</Warning>
