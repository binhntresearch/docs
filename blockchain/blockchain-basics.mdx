## Hash Function

<Callout type="info">
Hash function biến bất kỳ input nào thành output fixed-length (256-bit cho SHA-256). Không thể đảo ngược — và chỉ cần thay đổi 1 bit trong input sẽ tạo ra hash hoàn toàn khác.
</Callout>

```python
# Demo hash
import hashlib

data = "Hello, Blockchain!"
hash_result = hashlib.sha256(data.encode()).hexdigest()
print(f"Hash: {hash_result}")
# Hash: 3c4a16b8637a81b69a32c4e0d8f5a2b...

# Thay đổi 1 ký tự → hash hoàn toàn khác
data2 = "Hello, Blockchain?"
hash2 = hashlib.sha256(data2.encode()).hexdigest()
print(f"Hash2: {hash2}")
```

## Consensus Mechanisms

<Tabs>
  <Tab title="Proof of Work (PoW)">
    - Miner cạnh tranh giải câu đố toán học
    - Ai giải được trước → đào block tiếp theo → nhận reward
    - An toàn nhưng tốn năng lượng
    - Bitcoin vẫn sử dụng PoW
  </Tab>
  <Tab title="Proof of Stake (PoS)">
    - Validator stake ETH (khóa 32 ETH)
    - Validator được chọn ngẫu nhiên để xác thực block
    - Tiết kiệm 99.95% năng lượng so với PoW
    - **Ethereum đã chuyển sang PoS từ Sept 2022 (The Merge)**
  </Tab>
  <Tab title="Delegated PoS (DPoS)">
    - Token holder bầu delegate để xác thực
    - Delegate chạy node và xác nhận giao dịch
    - Nhanh hơn PoS truyền thống
    - EOS, Tron sử dụng DPoS
  </Tab>
</Tabs>
