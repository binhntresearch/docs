Claude Code là trợ lý lập trình AI chạy trực tiếp trong terminal, giúp bạn hiểu codebase, sửa lỗi, viết test và tạo pull request — tất cả chỉ bằng ngôn ngữ tự nhiên. Hướng dẫn này sẽ đưa bạn từ bước cài đặt đến hoàn thành tác vụ lập trình đầu tiên trong vòng 5 phút.

<Note>
  **Yêu cầu trước khi bắt đầu**
  - **Node.js 18+** — kiểm tra bằng lệnh `node --version`
  - **Tài khoản Claude** với gói subscription hoặc API key hợp lệ
  - **Git** (khuyến nghị) — cần thiết cho các lệnh tạo commit và PR
</Note>

## Các bước thực hiện

<Steps>

### Cài đặt Claude Code

Chọn phương thức cài đặt phù hợp với hệ điều hành của bạn:

<CodeGroup>

```bash macOS / Linux / WSL
curl -fsSL https://claude.ai/install.sh | bash
```

```powershell Windows PowerShell
irm https://claude.ai/install.ps1 | iex
```

```bash Homebrew
brew install --cask claude-code
```

```bash WinGet
winget install Anthropic.ClaudeCode
```

```bash npm (thay thế)
npm install -g @anthropic-ai/claude-code
```

</CodeGroup>

Sau khi cài đặt xong, xác nhận bằng lệnh:

```bash
claude --version
```

### Khởi động Claude trong dự án của bạn

Di chuyển vào thư mục dự án và khởi động Claude Code:

```bash
cd your-project
claude
```

Lần đầu chạy, Claude Code sẽ mở trình duyệt để bạn đăng nhập vào tài khoản Claude. Hoàn tất xác thực, quay lại terminal — bạn sẽ thấy giao diện chat sẵn sàng nhận lệnh.

<Tip>
  Nếu bạn dùng API key thay vì subscription, đặt biến môi trường trước khi chạy:
  ```bash
  export ANTHROPIC_API_KEY="sk-ant-..."
  claude
  ```
</Tip>

### Giao tác vụ đầu tiên cho Claude

Gõ trực tiếp yêu cầu của bạn bằng tiếng Anh hoặc tiếng Việt. Claude Code sẽ đọc các file liên quan, đề xuất thay đổi và hỏi xác nhận trước khi thực thi các lệnh có thể ảnh hưởng đến hệ thống.

Thử một vài ví dụ dưới đây để làm quen:

<Tabs>
  <Tab title="Khám phá codebase">
    ```bash
    claude "explain this codebase to me"
    ```
    Claude sẽ duyệt cấu trúc thư mục, đọc các file chính và trả lời tổng quan về kiến trúc, công nghệ sử dụng và luồng hoạt động của dự án.
  </Tab>
  <Tab title="Sửa lỗi">
    ```bash
    claude "fix the failing tests in the auth module"
    ```
    Claude tìm các test đang thất bại, phân tích nguyên nhân và đề xuất (hoặc áp dụng trực tiếp) bản sửa lỗi — kèm giải thích rõ ràng.
  </Tab>
  <Tab title="Viết test">
    ```bash
    claude "write tests for the payment service"
    ```
    Claude đọc code hiện tại của `payment service`, tạo bộ test coverage hợp lý và thêm vào đúng vị trí trong dự án.
  </Tab>
  <Tab title="Tạo Pull Request">
    ```bash
    claude "create a PR with my changes"
    ```
    Claude xem lại các thay đổi chưa commit, viết mô tả PR rõ ràng và chạy lệnh git để đẩy branch, tạo PR trên GitHub/GitLab.
  </Tab>
</Tabs>

### Nắm vững các lệnh quan trọng

Làm chủ các lệnh sau để làm việc hiệu quả hơn với Claude Code:

| Lệnh | Chức năng |
|---|---|
| `/help` | Hiển thị toàn bộ lệnh có sẵn |
| `/model` | Chuyển đổi model (ví dụ: Claude Sonnet, Opus) |
| `/compact` | Nén lịch sử hội thoại để tiết kiệm context |
| `/clear` | Xóa toàn bộ hội thoại, bắt đầu lại |
| `Ctrl+C` | Ngắt Claude khi đang thực thi |
| `Shift+Tab` | Chuyển đổi chế độ quyền (permission mode) |

<Note>
  **Chế độ quyền (Permission Modes)**

  Claude Code có ba chế độ kiểm soát mức độ tự động:
  - **Default** — Claude hỏi xác nhận trước khi chạy lệnh có thể gây ảnh hưởng
  - **Auto** — Claude tự động thực thi, phù hợp cho môi trường sandbox/CI
  - **Manual** — Claude luôn hỏi trước mọi hành động

  Nhấn `Shift+Tab` để chuyển đổi giữa các chế độ ngay trong phiên làm việc.
</Note>

</Steps>

## Tuỳ chỉnh Claude cho dự án của bạn

Tạo file `CLAUDE.md` tại thư mục gốc của dự án để cung cấp ngữ cảnh riêng — quy ước đặt tên, công nghệ sử dụng, các lệnh thường dùng, hay bất kỳ hướng dẫn nào bạn muốn Claude luôn ghi nhớ:

```markdown
# My Project

## Tech Stack
- Backend: Node.js + Express
- Database: PostgreSQL (ORM: Prisma)
- Testing: Vitest

## Conventions
- Use Vietnamese for comments and commit messages
- Always run `npm test` before suggesting a PR
- Follow the existing folder structure under `src/modules/`
```

Claude Code tự động đọc file này mỗi khi bạn khởi động trong thư mục đó.
