Mobile UI/UX khác web — **touch-first**, screen nhỏ, gestures phức tạp hơn. Hiểu mobile design patterns = app chuyên nghiệp hơn.

## Mobile Design Principles

<Check>
- **Thumb-friendly:** CTA buttons ở bottom third của screen
- **Minimal navigation:** Bottom tabs (≤5 items) hoặc hamburger menu
- **Clear hierarchy:** One primary action per screen
- **Consistent patterns:** iOS → Cupertino, Android → Material Design
- **Loading states:** Skeleton screens, shimmer effects
- **Offline-first:** Hiển thị cached data khi mất kết nối
</Check>

## Responsive Design

```tsx
// React Native — Responsive với Dimensions
import { Dimensions, useWindowDimensions } from 'react-native';

export function ResponsiveScreen() {
  const { width, height } = useWindowDimensions();
  const isTablet = width >= 768;

  return (
    <View style={{
      flexDirection: isTablet ? 'row' : 'column',
      padding: isTablet ? 24 : 16
    }}>
      <View style={{ flex: 1 }}>
        <Text style={{ fontSize: isTablet ? 24 : 18 }}>Title</Text>
      </View>
      <View style={{ flex: isTablet ? 1 : 0, width: isTablet ? '50%' : '100%' }}>
        {/* Content */}
      </View>
    </View>
  );
}
```

## Animations — Flutter

```dart
// Fade animation
class FadeIn extends StatefulWidget {
  final Widget child;
  const FadeIn({super.key, required this.child});

  @override
  State<FadeIn> createState() => _FadeInState();
}

class _FadeInState extends State<FadeIn> with SingleTickerProviderStateMixin {
  late AnimationController _controller;
  late Animation<double> _animation;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      duration: const Duration(milliseconds: 500),
      vsync: this,
    );
    _animation = CurvedAnimation(parent: _controller, curve: Curves.easeIn);
    _controller.forward();
  }

  @override
  Widget build(BuildContext context) {
    return FadeTransition(opacity: _animation, child: widget.child);
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }
}
```

## Touch Gestures

```tsx
// React Native — Swipe to delete
import { Swipeable } from 'react-native-gesture-handler';

function TaskItem({ task, onDelete }) {
  const renderRightActions = (progress, dragX) => (
    <TouchableOpacity style={styles.deleteAction} onPress={onDelete}>
      <Text style={styles.deleteText}>Delete</Text>
    </TouchableOpacity>
  );

  return (
    <Swipeable renderRightActions={renderRightActions}>
      <TaskCard title={task.title} status={task.status} />
    </Swipeable>
  );
}
```

## Platform-Specific Components

```tsx
// iOS vs Android differences
import { Platform, StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  container: {
    paddingTop: Platform.OS === 'ios' ? 44 : 20, // Status bar
  }
});

// Platform-specific components
import { Platform } from 'react-native';

function getHeaderStyle() {
  return Platform.select({
    ios: { shadowColor: '#000', shadowOffset: { width: 0, height: 2 } },
    android: { elevation: 4 }
  });
}
```
