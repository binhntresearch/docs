Web3 là thế hệ tiếp theo của Internet — nơi người dùng **sở hữu dữ liệu của mình**, không phụ thuộc vào các công ty tập trung như Google, Meta, hay Amazon.

## Web3 là gì?

<Callout type="info">
**Web1:** Read-only · Static websites · Mọi người đọc nội dung · **Web2:** Read + Write · Social media · Một vài công ty nắm data · **Web3:** Read + Write + Own · Blockchain · Người dùng sở hữu tài sản số
</Callout>

## So sánh 3 thế hệ Web

<Tabs>
  <Tab title="Web1 (1990s-2004)">
    - **Read-only**
    - Static websites, blog cá nhân
    - Ai cũng có thể đọc
    - Powered by HTML, CSS
    - Không có user-generated content
  </Tab>
  <Tab title="Web2 (2004-nay)">
    - **Read + Write**
    - Facebook, YouTube, TikTok
    - User tạo content nhưng platform sở hữu data
    - Tập trung hóa — bị kiểm duyệt, khóa tài khoản
    - Mô hình: attention = tiền
  </Tab>
  <Tab title="Web3 (2020+)">
    - **Read + Write + Own**
    - Blockchain thay thế công ty
    - User sở hữu tài sản (wallet, NFT, tokens)
    - Phi tập trung, minh bạch
    - Không cần trust platform
  </Tab>
</Tabs>

## Từ khóa chính

| Thuật ngữ | Ý nghĩa |
|---|---|
| **Wallet** | Ví lưu private key — giao tiếp với blockchain |
| **DApp** | Decentralized App — Frontend + Smart Contract |
| **DeFi** | Tài chính phi tập trung |
| **NFT** | Non-Fungible Token |
| **DAO** | Tổ chức tự quản |
| **DEX** | Decentralized Exchange |
| **L2** | Layer 2 — scaling solution (Optimism, Arbitrum) |
| **Account Abstraction** | Smart account thay thế EOA |

## Sự khác biệt giữa Blockchain và Web3

<Warning>
**Blockchain** = công nghệ nền tảng (data structure + cryptography)
**Web3** = hệ sinh thái ứng dụng xây trên blockchain

Blockchain là cơ sở hạ tầng. Web3 là những gì user thấy và dùng (UI/UX, apps, communities).
</Warning>
