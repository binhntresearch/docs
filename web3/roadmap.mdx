Lộ trình đầy đủ từ **Frontend Developer (Web2)** trở thành **Web3 Developer chuyên nghiệp**.

## Phase 1: Nền tảng vững chắc (Tuần 1-4)

<Check>
- React/Next.js cơ bản đến nâng cao
- TypeScript
- Web3 concepts (blockchain, wallet, DApp)
- Hiểu cách blockchain hoạt động
- Đọc Ethereum docs cơ bản
</Check>

## Phase 2: Web3 Frontend (Tuần 5-8)

<Check>
- **ethers.js / viem** — thư viện kết nối blockchain
- **wagmi** — React hooks cho wallet
- **RainbowKit / Web3Modal** — UI components kết nối ví
- **Account Abstraction** — ví thông minh
- Test với MetaMask và Sepolia
</Check>

## Phase 3: Smart Contract (Tuần 9-16)

<Check>
- Solidity cơ bản
- Hardhat / Foundry (development framework)
- OpenZeppelin contracts
- DeFi primitives (AMM, lending, staking)
- Security best practices
- Test coverage > 90%
- Audit Solidity code
</Check>

## Phase 4: Advanced (Tuần 17-24)

<Check>
- **The Graph** — index on-chain data cho DApp
- **IPFS / Arweave** — decentralized storage
- **ENS** — domain name (.eth)
- **Layer 2** — Optimism, Arbitrum, zkSync
- **Multi-chain** — viết contract chạy nhiều chain
- **Account Abstraction / ERC-4337** — smart wallet
</Check>

## Dự án thực chiến theo từng phase

| Phase | Dự án | Stack |
|---|---|---|
| 1 | Portfolio site connect wallet | React + wagmi |
| 2 | Token swap UI | Next.js + viem + RainbowKit |
| 3 | ERC-20 token + staking contract | Solidity + Hardhat |
| 4 | Decentralized Social Network | Ceramic + The Graph |

## Học ở đâu?

<Tip>
Nguồn học chất lượng cho Web3 Development:

- **Solidity:** Solidity Docs, Solidity by Example
- **Hardhat:** Hardhat Tutorial, Hardhat Starter Kit
- **wagmi:** wagmi.sh docs (rất chi tiết)
- **OpenZeppelin:** contracts.openzeppelin.com
- **Ethereum:** ethereum.org/developers
- **Practice:** Speedrun Ethereum (buildspace)
</Tip>
