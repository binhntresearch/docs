Web3 Social Networks cho phép user **sở hữu nội dung** của họ — không bị khóa tài khoản, không bị thao túng thuật toán.

## So sánh Web2 vs Web3 Social

| Web2 | Web3 |
|---|---|
| Platform owns user data | User owns content (NFT) |
| Algorithm controls feed | You choose your feed |
| Account suspension | Can't be censored |
| Monetization for platform | Monetization for creator |
| Ads-driven | Token-incentivized |

## Lens Protocol

Lens là giao thức social graph phi tập trung trên Polygon — user sở hữu profile NFT và content NFT.

### Profile NFT

```solidity
// User có 1 profile NFT duy nhất
const profile = await lensHub.createProfile({
  to: '0xUserAddress',
  handle: 'alice.lens',
  followModule: zeroAddress,
  followNFTURI: ''
});

// Set metadata cho profile
await lensHub.setProfileMetadataURI(profileId, 'ipfs://metadata...');
```

### Publish Post

```solidity
const post = await lensHub.post({
  profileId: profileId,
  contentURI: 'ipfs://QmPostContent...',
  collectModule: emptyCollectModule.address,
  collectModuleInitData: [],
  referenceModule: zeroAddress,
  referenceModuleInitData: []
});

// Post là NFT — user owns it
```

### Follow

```solidity
// Follow 1 profile
await lensHub.follow([profileToFollow, []]);

// Unfollow
await lensHub.unfollow(profileToFollow);
```

## Farcaster

Farcaster là social graph phi tập trung — user có **FID** (Farcaster ID).

```javascript
// Cast — message on Farcaster
import { createClient } from '@farcaster/hub-nodejs';

const client = createClient();

const cast = await client.submitCast({
  text: 'Hello Farcaster!',
  fid: 1234,
  network: 'mainnet'
});
```

## Content Moderation

<Tip>
Web3 social không có admin khóa tài khoản. Giải pháp: **Feed Filters** — user tự chọn follow ai, hiển thị gì. Community có thể tạo "moderation lists" chia sẻ.
</Tip>

## Discovery & Feeds

Cách user tìm content trong Web3:

| Thuật toán | Đặc điểm |
|---|---|
| **Chronological** | Mới nhất lên đầu |
| **Follow-based** | Chỉ từ người mình follow |
| **Interest-based** | Dựa trên likes, interactions trước |
| **Community-curated** | Lists từ community leaders |
| **Token-weighted** | Vote bằng token cho content chất lượng |
