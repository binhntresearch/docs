Account Abstraction (AA) cho phép tạo **Smart Account** — loại ví thông minh có thể lập trình được, thay thế ví EOAs truyền thống (MetaMask).

## Tại sao cần AA?

## Kiến trúc ERC-4337

```
┌─────────────────────────────────────────┐
│           User Operation                  │
│  (sender, callData, gasLimit, ...)       │
└──────────────┬───────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│  EntryPoint Contract (Singleton)          │
│  → Bundlers submit UserOps               │
└──────────────┬───────────────────────────┘
               ↓
┌─────────────────────────────────────────┐
│  Smart Account (per user)                │
│  → Verify signature, execute callData    │
└─────────────────────────────────────────┘
```

## Sử dụng với Permissionless.js

```bash
npm install permissionless viem
```

```javascript
import { createBundlerClient } from 'permissionless';
import { createSmartAccountClient } from 'permissionless/accounts';
import { createPimlicoPaymasterClient } from 'permissionless/paymaster';

// Bước 1: Tạo Smart Account từ owner
const owner = privateKeyToAccount(ownerPrivateKey);

const smartAccount = await createSmartAccountClient({
  chain: sepolia,
  owner,
  factoryAddress: '0xYourFactory', // SimpleAccount factory
  entryPoint: ENTRY_POINT_ADDRESS
});

// Bước 2: Send UserOp (có thể kèm paymaster cho gasless)
const bundler = createBundlerClient({
  chain: sepolia,
  transport: http('https://bundler.example')
});

const paymaster = createPimlicoPaymasterClient({
  transport: http('https://paymaster.example')
});

const userOpHash = await smartAccount.sendUserOperation({
  calls: [{
    to: '0xRecipient',
    value: 0n,
    data: '0x' // function call data
  }],
  paymaster: paymaster // Gas trả bởi DApp
});

await bundler.waitForUserOperationReceipt({ hash: userOpHash });
```

## Smart Account với Safe

```javascript
import { createSafeAccount } from 'permissionless/accounts/safe';

const safeAccount = await createSafeAccount({
  client: publicClient,
  owners: [owner],
  threshold: 2, // 2-of-3 multisig
  saltNonce: 0n
});
```

## Session Keys

```javascript
// Cấp session key cho Game — có thể gọi 1 contract cụ thể, hết hạn sau 1 ngày
const sessionKeyData = encodeFunctionData({
  abi: sessionManagerAbi,
  functionName: 'createSession',
  args: [{
    signer: gameContract,
    validUntil: Math.floor(Date.now() / 1000) + 86400, // 1 day
    permissions: [{
      contractAddress: nftContract,
      selector: 'mint(address)'
    }]
  }]
});

await smartAccount.execute([{
  to: sessionManager,
  data: sessionKeyData
}]);
```

## Stack AA Production

<Tip>
Production stack cho Account Abstraction:

- **Bundler:** Pimlico, Stackup, Alchemy
- **Paymaster:** Pimlico (hỗ trợ sponsorship)
- **Smart Account:** Safe (battle-tested) hoặc SimpleAccount
- **SDK:** permissionless.js, Biconomy SDK
</Tip>
